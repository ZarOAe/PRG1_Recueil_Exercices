# `while` (2)

Que va afficher à l'exécution chacun des groupes d'instructions ci-dessous ?

~~~cpp
// 1
int i = 0;
while (i <= 5) 
   i++;
cout << i;
~~~

6


<details>
<summary>Solution</summary>

~~~
6
~~~
</details>

~~~cpp
// 2
int i = 0, j = 0;
while (i <= 5) 
   i += j++;
cout << i << ' ' << j;
~~~

// i = 0 + 1; i=1 j=2
//i = 1 + 2; i=3 j=3;
//i = 3 + 3; i=6 j=4;
